
ToneLab 

**Live:** [tonelab.vercel.app](https://tonelab.vercel.app)

AI-powered music generation using Pollinations BYOP. Generate tracks with multiple models including AceStep (free), ElevenMusic, and Stable Audio.

## ✨ Features

-   **Multi-Model Support**: Switch between free and premium AI audio models
-   **Glassmorphism UI**: Modern dark theme with smooth animations
-   **Instant Playback**: Stream and download generated tracks in-browser
-   **Serverless Backend**: Secure API routing via Next.js

## 🛠️ Tech Stack

-   Next.js 14 (App Router)
-   TypeScript
-   Tailwind CSS
-   Pollinations BYOP API
-   Vercel
